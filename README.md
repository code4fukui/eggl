# eggl

Easy library for WebGL ([glutil](glutil.js), [fragment-shader tag](fragment-shader.js), [GPGPU.js](GPGPU.js)).

## Demo

- https://code4fukui.github.io/eggl/

## Features

- Provides utility functions for creating and managing WebGL elements
- Includes a custom `fragment-shader` HTML element for easy WebGL visualization
- Supports GPGPU (General-Purpose computing on Graphics Processing Units) with `GPGPU.js`

## Library

- [wgld.org | library: minMatrix.js リファレンス |](https://wgld.org/d/library/l001.html) → [mat4.js](mat4.js) where the `dest` parameter is moved to the front
- [glMatrix](https://glmatrix.net/) - a more comprehensive matrix calculation standard library than `mat4.js`
