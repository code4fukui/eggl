# eggl

WebGLを簡単に扱えるライブラリです。[glutil.js](glutil.js)、[fragment-shader タグ](fragment-shader.js)、[GPGPU.js](GPGPU.js)などのユーティリティ関数が提供されています。

## デモ
- https://code4fukui.github.io/eggl/

## 主な機能
- WebGLの基本的な機能を提供するユーティリティ関数
- WebGLのシェーダを簡単に扱えるカスタムタグ
- GPGPUを使ったデータ処理ライブラリ

## 必要環境
- WebGL2に対応したブラウザ

## 使い方
npmやその他の方法でインストールし、各ユーティリティ関数を呼び出して使用します。
