# eggl

- WebGLで簡単に作れるライブラリ ([glutil.js](glutil.js)、[fragment-shader タグ](fragment-shader.js)、[GPGPU.js](GPGPU.js))

## デモ

- https://code4fukui.github.io/eggl/

## 主な機能

- WebGLのユーティリティ関数
- WebGLのシェーダを簡単に扱えるカスタムタグ
- GPGPUを使ったデータ処理ライブラリ

## 必要環境

- WebGL2に対応したブラウザ

## 使い方

npmなどでインストールし、各機能を呼び出して使います。
